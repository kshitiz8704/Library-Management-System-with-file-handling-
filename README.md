# C++ File Handling Program

## Description
This repository contains a C++ program demonstrating file handling concepts such as reading from and writing to files.

## Files Included
- LMS.cpp – C++ source code
- library.txt – Input/output file used in the program

## How to Run
1. Compile the program:
   g++ main.cpp -o program
2. Run the program:
   ./program

Make sure `data.txt` is in the same folder.
